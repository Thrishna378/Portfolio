My Projects

Smart Assistive Wheelchair

A next-generation wheelchair designed to enhance mobility and safety for differently-abled individuals. The system integrates an ESP32 microcontroller with ultrasonic sensors for obstacle detection, an MPU6050 accelerometer for fall detection, and a buzzer/LED for real-time alerts.

IoT functionality allows remote monitoring, emergency notifications via Firebase, and data storage for caregiver access. The wheelchair also supports motor control through an L298N driver, making it both intelligent and responsive.

This project combines hardware, software, and IoT technologies to improve independence and safety for users.

CampusFind - Lost & Found System

A smart campus lost and found system that uses AI-powered matching to reunite students with their lost items. The system features secure login using campus credentials for trust and safety. Users can report lost/found items with details, images, location, and timestamp.

AI-based smart matching uses item description, category, color, location, and image similarity to find potential matches. Instant notifications alert users when matches are found, enabling secure in-app communication between finder and owner for efficient item recovery with verification.

This project leverages machine learning and real-time database technology to create a seamless lost and found experience for campus communities.
